# DEE 예제

이 디렉토리는 Digital Emotion Engine의 실제 동작을 보여주는 예제 코드를 포함합니다.

## 예제 목록

### 1. character_simulation.c

3명의 캐릭터(탐험가, 전략가, 수호자)에게 동일한 자극을 주고 각각의 감정 변화와 능동 발화를 비교합니다.

**실행 방법**:

```bash
# 컴파일
make

# 실행
./character_simulation
```

# 시나리오:

## 시간 자극 설명
2초 칭찬 "정말 잘했어!" → 긍정 반응
8초 위협 "이대로면 위험해!" → 부정 반응
15초 반복질문 "왜? 왜? 왜?" → 짜증 유발
22초 신뢰 "너를 믿어. 함께하자." → 사회적 유대

# 직접 만들어 보기

```c
#include "kc_hormone.h"
#include "kc_persona.h"

int main() {
    KcEmotionEngine *dee = kc_dee_create();
    kc_persona_apply(dee, KP_탐험가);
    
    // 1초에 한 번씩 감정 업데이트
    for (int i = 0; i < 10; i++) {
        kc_dee_tick(dee, 1.0f);
        kc_dee_print(dee);
    }
    
    kc_dee_destroy(dee);
    return 0;
}
```

## ✅ 이 예제의 장점

1. **실제 동작 확인**: DEE 엔진이 컴파일되고 실행되는 것을 즉시 확인 가능
2. **성격 차이 시각화**: 동일 자극에 대한 3가지 성격의 반응을 한눈에 비교
3. **능동 발화 시연**: AI가 스스로 말을 거는 장면을 실제로 확인
4. **확장성**: 사용자가 자신만의 시나리오를 쉽게 추가 가능
5. **교육용**: DEE의 핵심 개념(호르몬, 감정, 성격)을 직관적으로 이해
